# PlagAlert
[![Build Status][travis-image]][travis-url]
## Introduction
This project serves as a plagiarism detector for debate speeches using multiple state-of-the-art algorithm.

## Algorithm
| Algorithm      | Status |
| ----------- | ----------- |
| Levenshtein | Complete    |
| Cosine Similarity | Complete |
